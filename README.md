## 📖 Overview

StudyNotion is a full-stack EdTech platform built using the MERN Stack. It provides a seamless learning experience for students and allows instructors to create, manage, and share educational content.

---

## ✨ Features

### Student Features
- User Registration & Login
- Browse Available Courses
- Add Courses to Wishlist
- Purchase Courses
- Access Enrolled Courses
- Rate and Review Courses
- Manage Profile

### Instructor Features
- Create New Courses
- Upload Course Content
- Edit and Delete Courses
- View Course Analytics
- Manage Profile

---

## 🏗 System Architecture

StudyNotion follows a client-server architecture:

- Frontend built with React.js
- Backend developed using Node.js and Express.js
- MongoDB used for data storage
- Cloudinary used for media management
